# 模板字符串

# 模板字符串嵌套

let a = 1
let b = 2
let str = `--------${a + `${b}`}---------`
console.log(str) // output: 12

# 标签模板

const name = "HarrisonLee"
const age = 22
const msg = tag`姓名：${name}, 年龄：${age}`

    console.log(strings)
    console.log(vars)
}

strings是由${}分割后的字符串数组，而变长参数包含了标签字符串中的所有参数的值。可以利用该特性来对模板字符串中某些值进行处理。下面是一个简单的例子：

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  <div id="contents"></div>
  <script>
    const contents = [
      {brand: "宝马", content: "宝马的总裁跳槽到奔驰"},
      {brand: "奔驰", content: "奔驰的总裁跳槽到奥迪"},
      {brand: "奥迪", content: "奥迪的总裁跳槽到宝马"}
    ]
    const msg = `${contents.map(item => 
      highlight`<li>品牌: ${item.brand}, 内容：${item.content}</li>`).join("")}`
    function highlight(strings, ...vars) {
      return strings.map((str, key) => {
        return (str + (vars[key] 
        ? vars[key].replace('跳槽', `<a href="https://www.lagou.com">跳槽<a/>`) 
        : ""))
      }).join("")
    }
    console.log(msg)
    document.getElementById("contents").innerHTML = msg
  </script>
</body>
</html>

# 字符串截取

# substring()

string.slice(start[, end])
1

使用需要注意以下几点：

  • substring()从提取的字符start可达但不包括end。

  • 如果start等于end，substring()返回一个空字符串。

  • 如果end省略，则将substring()字符提取到字符串的末尾。

  • 如果任一参数小于0或是NaN，它被视为为0。

  • 如果任何一个参数都大于string.length，则被视为是string.length。

  • 如果start大于end，那么效果substring()就好像这两个点被交换了一样，例如:

    string.substring(1, 0) == string.substring(0, 1)
    1

# substr()

string.substr(start[, length])
1

使用需要注意以下几点：

  • substr()会从start获取长度为length字符（如果截取到字符串的末尾，则会停止截取）。

  • 如果start大于或等于字符串的长度，则substr()返回一个空字符串。

  • 若start为负数,则将该值加上字符串长度后再进行计算（如果加上字符串的长度后还是负数，则从0开始截取）。

  • 如果length为0或为负数，substr()返回一个空字符串。如果length省略，则将substr()字符提取到字符串的末尾。

# slice()

slice()方法返回一个索引和另一个索引之间的字符串， 和substring()类似。

str.slice(beginIndex[, endIndex])
1
  • 若beginIndex为负数，则将该值加上字符串长度后再进行计算（如果加上字符串的长度后还是负数，则视作0）。
  • 如果beginIndex大于或等于字符串的长度，则slice()返回一个空字符串。
  • 如果endIndex省略，则将slice()字符提取到字符串的末尾。如果为负，它被视为strLength + endIndex，其中strLength是字符串的长度（如果加上字符串的长度后还是负数，则视作0）。
  • 如果beginIndex大于endIndex，返回空字符串。

提示

这三个函数都不会改变原字符串，会在原字符串的副本上操作然后返回结果。
