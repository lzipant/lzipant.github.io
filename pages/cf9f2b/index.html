将二叉查找树调整平衡

此题对应了LeetCode第1382题。

另外，此题与第108题十分类似，请参考这篇博客。只不过108题是将有序数组转化为平衡二叉搜索树。如果没做过第108题，可能很容易把本题想复杂，比如我最开始想要么新建一棵树（刚好那天复习了一下AVL树的实现方式），要么就地调整。想了一下，重建的做法应该不太高效，就原地调整，我一开始的思路是后序遍历原来的二叉树，然后自底向上调整（旋转）。但最后，这种方法也未能成功地实现，原因是直接照搬AVL树的旋转思路，导致一些情况下不能正确实现。

本题相比第108题就多了一个求解中序遍历的过程，关于二叉树的遍历问题可以参考这篇博客。

class Solution {
    public TreeNode balanceBST(TreeNode root) {
        List<TreeNode>aws = new ArrayList<>();
        inOrder(root, aws);
        return dfs(aws, 0, aws.size() - 1);
    }

    private void inOrder(TreeNode root, List<TreeNode>aws) {
        Deque<TreeNode>stack = new ArrayDeque<>();
        TreeNode tmp = root;
        while(stack.size() > 0 || tmp != null) {
            while(tmp != null) {
                stack.addLast(tmp);
                tmp = tmp.left;
            }
            tmp = stack.pollLast();
            aws.add(tmp);
            tmp = tmp.right;
        }
    }

    private TreeNode dfs(List<TreeNode> list, int a, int b) {
        if(a <= b) {
            int index = (a + b) / 2;
            TreeNode node = list.get(index);
            node.left = dfs(list, a, index - 1);
            node.right = dfs(list, index + 1, b);
            return node;
        } else {
            return null;
        }
    }
}
