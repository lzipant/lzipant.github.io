JavaScript中的数组

本文总结了一些JavaScript中关于数组的常用操作，包括数组的创建，类型与检测和元素操作，排序和遍历等。

# 创建数组

let hd= ["hdcms"]
hd[3] = "houdunren"
console.log(hd) // 自动扩充 ["hdcms", undefined, undefined, "houdunren"] 

let array = new Array(6, 7, 9); // array.length === 3
array = new Array(6); // array.length === 6
array = Array.of(6); // array.length === 1

关于fill()：

/**
* start: 默认值为0
* end: 默认值为数组长度
* 如果start为负，将被转换为array.length + start
* 如果end为负，将被转换为array.length + end
* 如果start大于等于end,则没有任何效果
* 如果start超过length,则没有什么效果
* 如果end超过length，则只对数组内的有效数据有效果
*/
array.fill(value[, start[, end]])

# 数组排序

  • sort()，默认升序排序，可以提供自定义的策略，注意是对数据进行就地排序，而非先复制数组再排序。

# 数组遍历

  • for

  • for of：遍历的是值，如果用来遍历对象，则是对象属性的值。

  • for in：遍历的是索引，如果用来遍历对象，则是对象属性的键的名称。

  • foreach(item, index, target)：target是被遍历的对象。

迭代器操作：

  • array.keys()：返回数组下标相关的迭代器。

  • array.values()：返回数组元素相关的迭代器。

  • array.entries()：返回关于数组迭代相关的迭代器。

其他操作：

  • every(prediction)：该函数将断言应用在每个元素上，如果每个元素都返回true，那么最后结果为true；否则返回为false。

  • some(prediction)：该函数将断言应用在元素上，直到应用到某个元素返回true时。

  • filter(prediction)：过滤元素，返回新数组。

  • map(function)：映射，返回新数组。

  • reduce(function)：归约操作。
