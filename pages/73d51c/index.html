二叉树的遍历

本文所有的遍历方式都接受两个参数，数的根节点(node)和保存遍历结果的列表(aws)。

对于前序、中序、后序的非递归遍历方式的思路比较直接，所以不再做文字叙述。

代码中使用到的BinTreeNode的定义如下：

package tree;

import lombok.AllArgsConstructor;
import lombok.Getter;
import lombok.NoArgsConstructor;
import lombok.Setter;

@NoArgsConstructor
@AllArgsConstructor
@Getter
@Setter
public class BinTreeNode<T> {
    private T data;
    private BinTreeNode<T> left;
    private BinTreeNode<T> right;

    public BinTreeNode(T data) {
        this.data = data;
    }

    @Override
    public String toString() {
        return data.toString();
    }
}

private <T> void preTraversalRecursive(BinTreeNode<T> node, List<T> aws) {
    if (node == null) {
        return;
    }
    aws.add(node.getData());
    preTraversalRecursive(node.getLeft(), aws);

# 非递归

private <T> void preTraversalIterative(BinTreeNode<T> node, List<T> aws) {
    Deque<BinTreeNode<T>> deque = new ArrayDeque<>();
    BinTreeNode<T> tmp = node;
    while (Objects.nonNull(tmp)) {
        aws.add(tmp.getData());
        if (Objects.nonNull(tmp.getRight())) {
            deque.addLast(tmp.getRight());
        }
        if (Objects.nonNull(tmp.getLeft())) {
            deque.addLast(tmp.getLeft());
        }
        tmp = deque.pollLast();
    }
}

# 非递归

private <T> void inTraversalIterative(BinTreeNode<T> node, List<T> aws) {
    Deque<BinTreeNode<T>> deque = new ArrayDeque<>();
    BinTreeNode<T> tmp = node;
    while (deque.size() > 0 || Objects.nonNull(tmp)) {
        while(Objects.nonNull(tmp)) {
            deque.addLast(tmp);
            tmp = tmp.getLeft();
        }
        tmp = deque.pollLast();
        aws.add(tmp.getData());
        tmp = tmp.getRight();
    }
}

同样地，利用栈来模拟递归。对于任何一个非空节点，寻找到最左下角的非空节点，并将沿途所有的节点（包括起点和终点）加入栈中保存。然后在栈不空的情况下，弹出栈顶元素，并保存数据到结果中，然后让指针指向其右子节点。

注意外层while循环的条件，加入Objects.nonNull(tmp)是为了确保根节点的右子树能够被遍历。因为在根节点弹出栈后，栈就空了，不管右子树存不存在，都会结束循环。

# 非递归

private <T> void postTraversalIterative(BinTreeNode<T> node, List<T> aws) {
    Deque<BinTreeNode<T>> deque = new ArrayDeque<>();
    BinTreeNode<T> tmp = node;
    while (Objects.nonNull(tmp)) {
        aws.add(tmp.getData());
        if (Objects.nonNull(tmp.getLeft())) {
            deque.addLast(tmp.getLeft());
        }
        if (Objects.nonNull(tmp.getRight())) {
            deque.addLast(tmp.getRight());
        }
        tmp = deque.pollLast();
    }
    Collections.reverse(aws);
}

非递归的后序遍历稍微有些复杂，不过根据逆后序遍历和前序遍历的关系，然后结合前序遍历的思路，就很容易实现后序遍历了。

上面这颗二叉树的前序遍历为： 12354，后序遍历为：35241。逆后序遍历为：14253。发现逆后序遍历就是在前序遍历的基础上交换了左右子树的遍历顺序得来的。上面非递归的前序遍历还特地强调了左右子树的顺序，此时只需交换两条语句的顺序即可。

# 层序遍历

private <T> void levelTraversal(BinTreeNode<T> node, List<T> aws) {
    if (Objects.nonNull(node)) {
        Deque<BinTreeNode<T>> deque1 = new ArrayDeque<>();
        Deque<BinTreeNode<T>> deque2 = new ArrayDeque<>();
        deque1.addLast(node);
        BinTreeNode<T> tmp;
        while (deque1.size() > 0 || deque2.size() > 0) {
            if (deque1.size() == 0) {
                deque1.addAll(deque2);
                deque2.clear();
            }
            tmp = deque1.pollFirst();
            assert tmp != null;
            aws.add(tmp.getData());
            if (Objects.nonNull(tmp.getLeft())) {
                deque2.addLast(tmp.getLeft());
            }
            if (Objects.nonNull(tmp.getRight())) {
                deque2.addLast(tmp.getRight());
            }
        }
    }
}

使用两个队列deque1和deque2，deque1保存了当前遍历的这一层的节点，然后deque2保存下一层的所有节点，并按照先进先出的顺序。待当前这一层遍历结束时，将deque2保存的值转移到deque1中，然后清空deque2。当deque1和deque2都为空时，遍历结束。
